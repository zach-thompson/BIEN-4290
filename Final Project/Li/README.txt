*****README*****

Implements the Li et al. paper via a Qt Creator GUI.

Prompts the user to select a .tiff image to analyze
Prompts the user to select a directory to write the output to
Run button initiates the algorithm

Outputs file called "output.csv" to indicated directory
Out file contains the calculated coordinates of all cones in the selected image

Error checking coming in next update. (v1.1)
